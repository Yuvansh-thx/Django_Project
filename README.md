# 🍋 Little Lemon Restaurant Website

A responsive restaurant website built using **Django** as part of my backend web development learning. The project demonstrates Django fundamentals including URL routing, views, templates, static files, models, forms, and database integration.

---

## 📌 Project Overview

Little Lemon is a fictional Mediterranean restaurant. This project provides a functional website where users can browse restaurant information, view the menu, and interact with the application through Django-powered pages.

The primary objective of this project is to understand the Django framework and implement the MVC (Model-View-Template) architecture.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Yuvansh-thx/little-lemon-django.git
```

### 2. Navigate into the project

```bash
cd little-lemon-django
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

macOS/Linux

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Apply migrations

```bash
python manage.py migrate
```

### 7. Run the development server

```bash
python manage.py runserver
```

Open your browser and visit

```
http://127.0.0.1:8000/
```
